# React Essentials Assignment - Task Manager

## Overview

This project is a Task Manager application built using React, Context API, and useReducer. It demonstrates global state management without external libraries.

## Features

* Add new tasks
* Edit existing tasks
* Mark tasks as complete/pending
* Delete individual tasks
* Clear all tasks
* View task statistics
* Responsive user interface
* Context API for global state management
* useReducer for predictable state updates

## Technologies Used

* React
* Context API
* useReducer
* CSS3
* Vite

## Project Structure

* Components for UI rendering
* Context for global state management
* Reducer for task actions
* Dedicated styling files

## Reducer Actions

* ADD_TASK
* EDIT_TASK
* TOGGLE_TASK
* DELETE_TASK
* CLEAR_TASKS

## Installation

```bash
git clone <repository-url>
cd react-essentials-assignment
npm install
npm run dev
```
